**本算法手册的主旨**

根据“**二八法则**”的原理，以付出20%的时间成本，获得80%的刷题的收益，让那些想进互联网大厂或心仪公司的人少走些弯路。

**本算法手册的特点**
- 1️⃣金刀亲手制作的图解
- 2️⃣大厂高频题型
- 3️⃣算法归类/相似题目归类
- 4️⃣通用解题思路
- 5️⃣LeetCode VIP题型
